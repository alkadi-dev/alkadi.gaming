# **App Name**: GameVerse Catalog

## Core Features:

- Jotform Data Connector: Integrate with the Jotform API to automatically pull game entries, including titles, detailed descriptions, category assignments, images, and video URLs, keeping the catalog up-to-date.
- Interactive Game Gallery: A responsive, grid-based gallery showcasing all games with a prominent thumbnail and title, allowing for quick browsing and discovery.
- Dynamic Game Detail Pages: Dedicated pages for each game, presenting its comprehensive description, an image carousel, and embedded video content in a clean, media-centric layout.
- Category-based Navigation: Enable users to filter and browse games by predefined categories such as RPG, Race, Adventure, and Open World, using an intuitive navigation system.
- AI-Assisted Description Refinement: A generative AI tool that analyzes game descriptions fetched from Jotform and refines them to improve clarity, engagement, and marketing appeal while retaining the original content's essence.

## Style Guidelines:

- The visual design will adopt a dark color scheme to make game media pop. Primary color is a deep, energetic purple (#732EB8), evoking digital creativity and depth. This choice is derived from HSL 270, 60%, 45%.
- The background color is a very dark, subtle violet-grey (#1A161D), providing a calm and immersive backdrop that aligns with the primary hue while ensuring excellent content visibility. This color is derived from HSL 270, 15%, 10%.
- The accent color is a vibrant, clear blue (#75A1FF), chosen to provide contrast and highlight interactive elements like buttons and selected categories. This color is analogous to the primary and is derived from HSL 240, 70%, 65%.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, techy, and engaging feel. Body text will utilize 'Inter' (sans-serif) for its readability and neutral, objective aesthetic, suitable for detailed descriptions.
- Minimalistic and sharp, outline-style icons will be used for navigation, search, and category filtering, ensuring a clean and contemporary visual language that complements the overall dark theme.
- The catalog will feature a responsive, card-based grid layout for game listings, optimizing for various screen sizes. Game detail pages will adopt a clean, single-column design with media (images, videos) given prominent space for an immersive viewing experience.
- Subtle and smooth transition animations will be used when navigating between the catalog and individual game pages. Hover effects on game cards will provide gentle visual feedback without distraction, enhancing interactivity.
